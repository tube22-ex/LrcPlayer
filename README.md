# LrcPlayer
data => song
songファイルの中にlrc形式の歌詞・よみファイルを入れて下さい。
【ファイル名形式】
歌詞　id_タイトル.lrc
よみ id_（読み）タイトル.lrc
URL id_youtubeURL.txt
【ファイル内容】

  -1_歌詞.lrc-
[00:00.00]
[00:03.53]歌詞
[00:05.10]

  -1_（読み）歌詞.lrc-
[00:00.00]
[00:03.53]かし
[00:05.10]

  -1_youtubeURL.txt-
https://www.youtube.com/watch?v=mpIUFEXgTtA
